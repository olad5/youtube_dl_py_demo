# youtube_dl python module demo



A Quick demo of how to use the youtube_dl module in python

## Installation


```bash
    git clone url 

    #Don't have pipenv installed; run
    pip install pipenv

    cd youtube_dl_py_demo

    #Install dependencies
    pipenv install


```
